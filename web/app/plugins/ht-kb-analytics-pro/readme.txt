Heroic Knowledge Base Analytics Pro enables analytics functionality for the Heroic Knowledge Base

== Description ==

Gain useful insights from your knowledge base with Heroic Knowledge Base Analytics Pro

== Installation ==


### INSTALL ANALYTICS PRO MANUALLY

1. Upload the unzipped "ht-kb-analytics-pro" folder to your websites /wp-content/plugins/ directory;
2. Activate the Heroic Knowledge Base Analytics Pro plugin through the "Plugins" page in WordPress;
3. Go to "after activation" below.

### AFTER ACTIVATION

The analytics screen can be accessed in the WordPress Admin Dashboard from Knowledge Base > Analytics.


== Frequently Asked Questions ==

= Who should use Heroic Knowledge Base Analytics Pro? =

This plugin is packaged an automatically enabled for eligible customers of Heroic Knowledge Base and/or the KnowAll knowledge base theme. This standalone plugin can be used where automatic installation and activation is not possible. The analytics screens within the knowledge base administration screens are then enabled.
