== Description ==

**Speed up your website with BunnyCDN** Content Delivery Network. This plugin allows you to easily **enable BunnyCDN on your WordPress** 
and enjoy greatly improved loading times around the world. Even better, it takes just a minute to set up, just enter the name of your Pull Zone and you're all set.

= How does it work? =
The plugin will automatically configure your website to replace existing static content links with the CDN links, greatly speeding up your content.

= Features =
* Replace static links with CDN links
* Automatic HTTPS configuration 
* Include or exclude specific directories or phrases
* Set a custom CDN hostname

= System Requirements =
* PHP >=5.3
* WordPress >=3.8
